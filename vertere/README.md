## 1. 概要説明
【タイトル】vertere  

【ジャンル】2Dアクションゲーム  

【難易度】普通~少し難しい  

【プレイ時間】3～10分程度  

【推奨年齢】全年齢  

所属サークルで行われた1weekゲームジャムで制作した2Dアクションゲームです。  

ジャンプをするたびにブロックのオン・オフが切り替わります。  

コインを集めて素早くクリアすることが目的となるゲームです。  

## 2. 制作期間
2024年 9/24~9/30 (1週間)
