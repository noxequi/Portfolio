## 1. 概要説明
【タイトル】vertere  

【ジャンル】2Dアクションゲーム  

【難易度】普通~少し難しい  

【プレイ時間】3～10分程度  

【推奨年齢】全年齢  

所属サークルで行われた1weekゲームジャムで制作した2Dアクションゲームです。  

ジャンプをするたびにブロックのオン・オフが切り替わります。  

コインを集めて素早くクリアすることが目的となるゲームです。

## 5. 遊び方・操作方法
ジャンプをするたびに、一部の足場のON・OFFが切り替わります。  
落下してしまったり、トゲにぶつかってしまうと残機が１減ってしまい、残機が尽きてしまうとゲームオーバーとなります。  
コインを集めるとクリア時のスコアが増加し、残り時間によってもスコアは変化します。  

### 操作方法:  
WASD/左右キー：移動  
スペースキー：ジャンプ  
左シフト+スペースキー：小ジャンプ  

## 6. 意識したこと
学祭で展示するにあたって、小さい子供や年配の方がよく遊びに来るため、操作方法がわかりやすいように、常に画面に操作方法を表示し、ゲーム開始時に、キーボードと操作を対応付ける図を表示し、操作方法を理解しやすいようにしました。
