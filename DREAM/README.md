## 1. 概要説明
【タイトル】DREAM  

【ジャンル】3Dステルスタワーディフェンスゲーム  

【難易度】普通   

【プレイ時間】10～30分程度  

【推奨年齢】全年齢  

CAPCOM GAMES COMPETIRIONにて制作したゲームです。  

舞台は夢の中の世界。  

プレイヤーは敵である「エトロス」から心のコアである「桜の木」を守りつつ、「エトロス」の発生源である「花」を破壊して夢の世界からの脱出を目指します。  

## 2. 制作意図
チーム制作を行いたいと思い、コンペティションに参加しました。 

チームで意見を出し合い、既存のゲームジャンルで今までなかったジャンルである、タワーディフェンスとステルスゲームを掛け合わせたゲームを作り、独創性を出していこうと考え制作を始めました。

## 6. 遊び方・操作方法
タワーディフェンスゲームでありながら、プレイヤーは自身でステージを駆け回り「花」の破壊を目指します。　　

倒した「エトロス」は一定確率で味方となり、召喚することができます。「桜の木」を守ったり、「花」を破壊しに行く補助として使えます。　　

### 操作方法:
(Xboxコントローラー想定)  
Rスティック：移動  

Lスティック：カメラ操作  

メニューボタン：メニュー

Aボタン：ジャンプ  

Bボタン：決定  

Yボタン：マップ  

Xボタン：攻撃  

RB / RT: 回避  

LB / LT: パレット表示  

パレット表示中にLスティック: 依代選択  

依代選択中にB: 召喚する依代決定  

## 7. 意識したこと・工夫点
接敵状況に応じてBGMが自然に切り替わるようにして、視覚情報だけでなく、音でも現在の危険度を判断できるようにしました。  

また、ゲームオーバーからリトライまでの流れや、シーン遷移をスムーズに行えるよう意識しました。

## 8. 改善点・反省点
初めて触るゲームエンジンだったため、仕様を確認しながら制作を進めていくこと、学業との両立からうまく制作を進めることができず、完成品が当初の予定していたものより縮小してしまいました。  

また、個人製作の延長のようになってしまい、各々がどのように実装しているかを把握することができず、合わせるときに苦労してしまいました。
