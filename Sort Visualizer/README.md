## 1. 概要説明  
このアプリケーションは、ソートアルゴリズムの可視化を行うアプリケーションです。  
大学の講義において、選択課題として実装しました。  
このアプリケーションでは、昇順、降順、ランダムのいずれかの順番で初期化された最大512個の配列の要素を並び替える過程を各ソートアルゴリズムにおいて可視化を行うことができます。  
実装されているアルゴリズムは、バブルソート、選択ソート、挿入ソート、シェルソート、ビンソート、クイックソート、マージソート、ヒープソートの8種類あり、各ステップを確認することができます。  
また、要素の小さいものから要素の大きいものまでのグラデーションを設定でき、背景色も変えることができます。また、任意のソートをLuaを用いて登録することができます。

## 5. 遊び方・操作方法
画面右側にある設定欄から、要素数、初期配列、ソートアルゴリズム、背景色等を選択し、Startボタンを押すことで設定が完了します。設定欄にある再生ボタンを押すと、ソートが開始されます。

### 操作方法:  
左クリックで選択  

## 6. 意識したこと
各ソートアルゴリズムにおけるソートの過程を、1ステップ事に確認することができるため、より詳細にソートの過程を見ることができるようにしました。また、Luaを用いてプレイヤーは任意のソートアルゴリズムを実装することができるようにしました。
